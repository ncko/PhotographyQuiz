# Photography Quiz

A quiz for new photographers who would like to test their knowledge.

Visit https://ncko.github.io/PhotographyQuiz/ to take the quiz.

## Created with
* jQuery
* Custom CSS
* Custom state management module called QUIZ
* Custom design using sketch
